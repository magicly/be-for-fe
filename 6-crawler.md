# 网络爬虫

## 爬虫原理

web 是一个由 url 链接起来的图， 爬虫就是遍历这个图。

非通用搜索引擎（Google、Bing、百度等），一般都是爬取特定内容， 可以有针对性地分析内容结构， 通过 Dom 分析、网络抓包、Headless 浏览器等进行爬取， 简单高效。

完整的爬虫涉及到很多东西， 包括 IP 代理池、爬取频率、验证码、登录验证等等， 网上有很多这方面的资料， 大家自行搜索。
